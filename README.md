# Personal Finance Tracker

A simple command-line Python program that helps you track income, expenses, estimated taxes, and simulate savings over time.  
The program saves your financial data in a JSON file and can generate a pie chart of your expenses as a PNG.

---

## Features

- Record monthly income and expenses  
- Automatic calculation of:
  - Total expenses  
  - Remaining balance  
  - Estimated tax (progressive brackets)  
- Save and load data from `financial_data.json`  
- Generate an **expense pie chart** (`expenses_chart.png`)  
- Simulate savings growth over multiple months  
- Input validation for all numeric values
